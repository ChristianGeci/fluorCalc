# fluorCalc
Python module to calculate expected count rate in Fluorescence XAFS.



# Tutorial
A tutorial is located in `notebooks/user_notebook.ipynb`



# Installation
To install with `pip`, run `py -m pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple/ fluorCalc` on Windows or `python3 -m pip install --index-url https://test.pypi.org/simple/ --extra-index-url https://pypi.org/simple/ fluorCalc` on Max/Linux
